# Resume Architect

A React app for building academic, technical, and creative resumes from a guided form and exporting them as PDF.

## Features

- A primary academic template modeled on the uploaded CV structure
- Three additional templates for executive, modern, and creative presentations
- Optional personal statement section
- Section visibility toggles
- Live preview with browser auto-save
- PDF export with multi-page support

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
